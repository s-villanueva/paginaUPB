Computing for Bioengineering

The use of computational tools has become a widely required competency across all branches of engineering, and is now considered a fundamental pillar in the training of every engineer. This course seeks to provide future bioengineers with a fundamental understanding of computers and mastery of programming, taking into account the problems they will encounter throughout their careers and professional lives. This theoretical-practical course not only introduces the fundamentals of computational systems, computational thinking, and programming, but also correctly guides students in the use of the Python programming language—one of the most important and widely used languages today—while learning to apply these tools effectively to problems related to bioengineering and bioinformatics.

Learning Objectives

  • Understand fundamental principles of Boolean logic and its application in digital circuits and biological models
  • Master binary representation of numerical data and basic arithmetic operations in computational systems
  • Develop computational thinking skills to design algorithms using pseudocode and flowcharts
  • Implement Python programs using control structures, functions, and data structures applicable to bioengineering problems
  • Apply error detection and correction codes in biological contexts, such as DNA sequences
  • Use data processing tools (NumPy, Pandas, SciPy) and regular expressions for preprocessing and analysis
  • Generate effective visualizations of scientific data using Matplotlib and Seaborn
  • Integrate knowledge of computational architecture and programming to solve multidisciplinary problems

Course Outline

    Topic 1: Computer Architecture
    • Introduction to Boolean logic
    • Binary representation of numbers
    • Introduction to computer arithmetic
    • Logic gates and circuits, truth tables

    Topic 2: Introduction to Computational Thinking
    • Basic concepts of computation
    • Algorithms and problem solving
    • Pseudocode and flowcharts
    • Data types and arrays

    Topic 3: Python Programming
    • Introduction to Unix prompt and related tools
    • Anaconda distribution, VSCode, and Jupyter notebooks
    • Basic Python syntax
    • Decision structures (if-else, if-elif-else)
    • Repetition structures (for, while loops)
    • Functions in Python, abstraction and modularity
    • Data structures: sets, lists, tuples, dictionaries

    Topic 4: Error Coding
    • Error detection codes (parity codes)
    • Error correction codes (Hamming, Reed-Solomon)
    • Convolutional codes
    • Applications to DNA sequences

    Topic 5: Processing Tools in Python
    • NumPy for array and matrix operations
    • Pandas for data management and analysis
    • SciPy for scientific analysis and differential equations
    • Regular expressions for pattern matching

    Topic 6: Visualization Tools in Python
    • Matplotlib for basic and scientific plots
    • Seaborn for advanced statistical visualizations
    • Data interpretation and communication



GPU acceleration transforms bioinformatics workflows by enabling real-time processing of massive genomic datasets, parallel execution of sequence alignment algorithms, and rapid training of machine learning models for protein structure prediction. Students can leverage GPU infrastructure to perform BLAST searches on entire genomes in seconds, execute molecular dynamics simulations with millions of atoms, and train deep learning models for gene expression analysis. With campus GPU resources, learners can implement GPU-accelerated libraries like CuPy and RAPIDS for bioinformatics, process next-generation sequencing data at unprecedented speeds, and develop computational pipelines for personalized medicine. This computational power prepares students for modern bioengineering where AI-driven drug discovery, genomic analysis, and systems biology are revolutionizing healthcare and biotechnology.
