Electromechanics Projects

The Electromechanics Project challenges students to integrate mechanical, electrical, and software engineering principles. These projects focus on developing practical, cost-effective solutions for healthcare, agriculture, and diagnostics using modern edge computing and AI technologies.

Research Proposals

medical_services

Low-cost Edge Computing Ultrasound Imager using AI

Development of a portable, affordable ultrasound device that processes imaging data locally on edge hardware. This project aims to democratize medical diagnostics in remote areas by using AI to enhance image quality and assist in interpretation without needing cloud connectivity.

agriculture

Determination of Customized Spectral Indexes for Disease Vegetation Classification

Research into creating specific spectral signatures to identify plant diseases early. By analyzing multispectral data, this system will define custom indexes to classify vegetation health more accurately than standard NDVI, enabling precision agriculture interventions.

visibility

Eye Disease Detection using Webcam Images

A computer vision system designed to screen for common eye conditions using standard, low-resolution webcam imagery. This project focuses on making ophthalmological screening accessible by training robust models to detect signs of cataracts or other pathologies from simple video feeds.
