Big Data

Big Data technologies enable organizations to process and analyze massive volumes of data that traditional systems cannot handle. This course covers the complete Big Data ecosystem, from distributed storage systems like Hadoop HDFS to real-time processing frameworks like Apache Spark and Kafka. Students will master data ingestion, storage, processing, and visualization at scale while working with cloud platforms and modern data lakes. The course emphasizes practical implementation using Python, Spark, and cloud services, preparing students to design and deploy scalable data pipelines for analytics, machine learning, and business intelligence.

Learning Objectives

  • Understand the fundamentals of distributed computing and Big Data architectures
  • Master Hadoop ecosystem: HDFS, MapReduce, YARN
  • Implement data processing pipelines using Apache Spark
  • Build real-time streaming applications with Kafka and Spark Streaming
  • Design and implement data lakes and data warehouses
  • Apply NoSQL databases: MongoDB, Cassandra, HBase
  • Develop scalable ETL pipelines for data integration
  • Deploy Big Data solutions on cloud platforms (AWS, Azure, GCP)

Course Outline

    Big Data Fundamentals:
    • Introduction to Big Data: 5 Vs (Volume, Velocity, Variety, Veracity, Value)
    • Distributed computing concepts
    • CAP theorem and data consistency
    • Big Data architectures and design patterns

    Hadoop Ecosystem:
    • HDFS: distributed file system
    • MapReduce programming model
    • YARN resource management
    • Hive and Pig for data processing

    Apache Spark:
    • Spark architecture and RDDs
    • DataFrames and Spark SQL
    • Spark MLlib for machine learning
    • Spark Streaming for real-time processing

    NoSQL Databases:
    • Document stores: MongoDB
    • Column-family stores: Cassandra, HBase
    • Key-value stores: Redis
    • Graph databases: Neo4j

    Stream Processing and Cloud:
    • Apache Kafka for event streaming
    • Real-time analytics and dashboards
    • Data lakes and lakehouses
    • Cloud Big Data services (AWS EMR, Azure HDInsight, GCP Dataproc)



GPUs allow for the implementation of hardware-accelerated data analysis workflows, processing massive volumes of information interactively for decision-making. Students will leverage GPU-accelerated frameworks like RAPIDS to perform distributed data processing at unprecedented speeds, execute complex analytics queries on terabyte-scale datasets in seconds, and train machine learning models on Big Data without sampling. With campus GPU infrastructure, learners can experiment with GPU-accelerated Spark, implement real-time analytics dashboards processing millions of events per second, and develop scalable data science pipelines that combine Big Data technologies with deep learning for advanced analytics applications.
