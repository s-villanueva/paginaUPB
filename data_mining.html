Data Mining

This course focuses on techniques and algorithms for extracting useful knowledge from databases. It begins with an introduction to the CRISP-DM methodology and handling different types of data, as well as key preprocessing processes such as cleaning, normalization, and dimensionality reduction. Students will study methods for extracting association rules, sequence mining, and graph mining, as well as supervised classification techniques and model evaluation using standard metrics. Unsupervised clustering techniques and their validation are also covered. Finally, time series analysis and predictive models are introduced. This course provides the foundation for transforming data into valuable knowledge, supporting decision-making in various fields.

Learning Objectives

  • Understand the role of data mining in the current context of data science.
  • Apply formal methodologies and techniques for extracting useful knowledge from datasets.
  • Define and formally describe different types of data, algorithms, and models used in data mining.
  • Recognize limitations and challenges in data analysis, as well as in the interpretation and validation of results.
  • Apply data preprocessing techniques including cleaning, normalization, and dimensionality reduction on real datasets.
  • Design and evaluate association rules, interpreting metrics such as support, confidence, and lift.
  • Implement and compare supervised classification methods in practical problems, analyzing their performance using evaluation metrics.
  • Apply clustering techniques to segment unlabeled data and validate results with appropriate indices.
  • Perform time series analysis evaluating their performance for prediction and decision-making.

Course Outline

    Topic 1: Introduction to Data Mining and Preprocessing
    • Introduction to Data Mining and CRISP-DM
    • Types of data: categorical, numerical, ordinal, text
    • Data cleaning and sampling
    • Normalization, discretization, and dimensionality reduction (PCA)
    • Data visualization and preparation for algorithms

    Topic 2: Pattern Discovery and Association Rules
    • Apriori algorithm and frequent rules
    • FP-Growth and metrics: support, confidence, lift, χ²
    • Sequence mining: concepts, GSP
    • PrefixSpan algorithm
    • Graph mining: basic gSpan

    Topic 3: Data Classification (Supervised)
    • Introduction to classification: objective, types, general flow
    • Naive Bayes: theory, application, and cases
    • Evaluation: Confusion Matrix, Precision, Recall, F1, ROC Curve, and AUC
    • Support Vector Machines (SVM)
    • Logistic Regression (Logit)
    • Decision Trees and Random Forest

    Topic 4: Unsupervised Clustering Methods
    • Similarity measures, K-means algorithm, SOM
    • Hierarchical clustering: agglomerative and divisive
    • DBSCAN: noise detection and arbitrary clusters
    • Cluster validation: Silhouette, Davies-Bouldin

    Topic 5: Time Series Analysis
    • Introduction to time series: trend, seasonality
    • ARIMA (autoregressive model) and SARIMA
    • Temporal model evaluation
    • Project preparation



Integrating GPU-accelerated AI tools into this Data Mining course significantly enhances the learning experience by enabling students to process large-scale datasets efficiently, train complex machine learning models in real-time, and experiment with deep learning architectures for pattern recognition. GPU computing allows for rapid execution of computationally intensive algorithms such as PCA, SVM, Random Forest, and neural networks. Students can leverage campus GPU infrastructure to work with big data frameworks, perform parallel processing for clustering algorithms, and explore advanced techniques like graph neural networks and transformer models for sequence mining. AI-driven analytics tools can provide intelligent insights, automated feature engineering, and real-time visualization of mining results, preparing students for modern data science challenges in industry and research.
